# OpenClaw 简体中文语言包

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![OpenClaw Version](https://img.shields.io/badge/OpenClaw-≥1.0.0-blue)](https://openclaw.ai)

为OpenClaw提供完整的简体中文界面本地化支持。

## 功能

- 核心界面元素中文本地化
- 命令响应消息中文翻译
- 错误消息中文版本
- 工具提示和状态信息本地化
- AI交互提示中文化

## 安装

### 方法1：通过OpenClaw插件管理器安装
```bash
openclaw plugins install https://github.com/openclaw-community/openclaw-lang-zh-CN
